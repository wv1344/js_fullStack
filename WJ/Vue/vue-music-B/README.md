## 简易音乐播放器

- clone网易云音乐接口项目到本地
  项目地址：https://github.com/Binaryify/NeteaseCloudMusicApi

- 进入网易云音乐项目地址
  npm install
  运行 node app.js 命令

- clone本项目到本地
  npm install
  通过 npm run dev 将项目运行起来
